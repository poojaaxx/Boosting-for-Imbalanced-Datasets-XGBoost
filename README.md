# Boosting for Imbalanced Datasets with XGBoost

This project demonstrates handling imbalanced classification problems using XGBoost.

## Key Features
- Exploratory Data Analysis on imbalanced data
- Baseline Logistic Regression model
- Class imbalance handling using SMOTE
- Advanced boosting with XGBoost
- Evaluation using PR-AUC and ROC-AUC
- NLP integration using TF-IDF

## Tools Used
Python, scikit-learn, XGBoost, imbalanced-learn, matplotlib
